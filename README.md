<h1>GavinWare V1</h1>
<p>A parody game of one of Nintendo/Intelligent System's popular games: WarioWare.</p>
<h1>Installation</h1>
<h2>Windows</h2>
<h3>If you don't have python installed:</h3>
<p>1. Go to python.org, then select Downloads</p>
<p>2. Select <q>get the standalone installer for Python 3.X.X</q></p>
<p>3. On the installer check the box <q>Add Python 3.X to PATH</q></p>
<p>4. Click on <q>Install Now</q> and follow further default instructions</p>
<h3>If you don't have the required dependencies:</h3>
<p>1. Extract the GavinWare ZIP file</p>
<p>2. Open terminal (or Powershell) and type the command <pre>
  <code>
    pip install -r /Downloads/GavinWare/requirements.txt
  <code>
</pre></p>
